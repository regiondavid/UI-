### 使用方法
1. 为要使用的输入框添加 `class="getDate"`
2. 引入 dist 下的 boundle.js（目前为uglify之后的）
3. 目前为了项目开发不额外去引入JQ，已经把原有的amazeUI的部分样式提取到style.css里面去了，所以目前该组件已经能够独立引用使用了。

### 目录说明
dist: webpack打包后的文件

js/css: 项目源码
